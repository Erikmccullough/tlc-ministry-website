# Fonts

**Primary UI (sans):** System UI stack — fast, elegant, no external request.  
`ui-sans-serif, system-ui, -apple-system, Segoe UI, Roboto, Helvetica, Arial`

**Optional display (headlines):** Consider pairing with one of:
- *Fraunces* (serif, soft contrast) — Ritual headings
- *Space Grotesk* (sans, geometric) — Titles, buttons
- *EB Garamond* (serif) — Liturgical text

**Monospace (code drift):**  
`ui-monospace, SFMono-Regular, Menlo, Consolas, "Liberation Mono", monospace`

**Usage**  
- Keep lines short (max 65ch) for oracular cadence.  
- Use **small-caps** for sigil names.  
- Avoid heavy weights; let glow & spacing carry the magic.
